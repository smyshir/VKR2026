# VKR2026

Обновлённый прототип ИС оценки координат источника излучения.

## Реализовано
- Потоковая обработка: данные пакетами 50/100/250/1000/произвольный размер.
- Предобработка датасета: удаление пропусков, слабых по весу, выбросов; при большом числе выбросов — восстановление интерполяцией.
- Непрерывный контур: данные -> БД -> предобработка -> расчет по батчам -> БД -> итоговый центр/радиус.
- 2 режима: `practical`, `experimental` (с эталонной точкой и расчетом ошибки).
- Рекомендация средства поражения по радиусу разброса.

## API
- `POST /api/measurements/batch`
- `POST /api/measurements/preprocess`
- `POST /api/stream/run`

## Запуск
```bash
pip install -r backend/requirements.txt
uvicorn backend.app.main:app --reload
```
